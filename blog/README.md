# Bhai Blog

## Installation
* Execute this command to install Gems:
```bash
    bundle install
```
* Or install it yourself as:
```bash
    gem install blog
```
* To run the Jekyll Server on the LocalHost, use:
```bash
bundle exec jekyll s --watch
```

## Usage
* This repository is primarily built by using the following libraries:
	* MaterializeCSS
	* jQuery
